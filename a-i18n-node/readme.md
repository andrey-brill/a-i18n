## a-i18n-node

> Core for NodeJs

## Install

```
$ npm install a-i18n-node
```

### CLI examples

- `npm link` && `a-i18n "I18n({ directory: './tests/huge', debug: true }) & load() & export({ type: 'custom' })"`
- `npm run cli load() & export({ type: 'custom' })"`
- `node ./cli.js "I18n({ autoExport: true }) & load()"`
